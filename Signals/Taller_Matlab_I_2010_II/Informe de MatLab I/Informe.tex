\section{Primer punto}
\noindent
EL primer punto de este taller recibe cualquier vector de cualquier tamaño es decir
$$XV = [a_1 ,a_2 ,a_3 ,...,a_{n - 1} a_n ]$$
es necesario escribirlo en esta sintaxis para que pueda ser reconocido por el programa, esto se hace en la ventana principal de MatLab después de inicializar el programa.\\

\subsection{Vector J}
\noindent
La función a graficar y analizar es

\begin{equation}
	y[n] = \sum\limits_{n - n_0 }^{n + n_0 } {x[k]}
		\label{ecu1}
\end{equation}

\noindent
EL valor de $n_0$ es ingresado por el usuario al igual que el vector de entrada.\\
Si el vector de entrada es $\delta [n - 3]$, al evaluarlo en la ecu. \ref{ecu1}, solo se vera un pulso en la posición (del gráfico) en $-3$, como es una sumatoria va a dar como resultado $1$, y este resultado se encuentra en este caso en $-3$.\\
El valor de $n_0$ tiene que ser un número entre $0 \le n < \infty$.\\
Para cada $YV[i]$, se debe de sumar todos los valores de $XV[n]$, definidos sobre el intervalo, como se han definido dos variables se deben utilizar dos \textit{for} anidados, el primero recorre todo el vector de entrada, mientras que el segundo recorre el intervalo de la \textit{suma}, que para este caso en especial es el intervalo de la \textbf{sumatoria}, es acumulada por la suma de los datos en la variable suma, además como esta en un for cuando termine de recorrer el intervalo de la sumatoria asigna el valor de la suma en el vector $S$ siendo este la salida que se va a graficar.

	y[n] = \left\{ {\begin{array}{*{20}c}   0 & {x[n] < 0}  \\

   \end{array}} \right\}

\noindent
La ecu. \ref{ecu2} recorre todo el vector de entrada (XV) por medio del comando for, iniciando la variable suma1, al igual que el anterior recorre nuevamente el vector tomando en cuenta la condición ${x[n] + x[n - 2]}$ para ${x[n] \ge 0}$ en caso contrario asignara valores los valores para que se cumpla la condición $0$ para ${x[n] < 0}$.

\section{Punto tres}
\noindent
La función a analizar es
\begin{equation}
	A\frac{{dy\left( t \right)}}{{dt}} + By\left( t \right) = C\frac{{dx\left( t \right)}}{{dt}} + Dx\left( t \right)
	\label{ecu7}
\end{equation}
\noindent
El usuario define los valores de A, B, C, D, $\delta$ y XV (vector de entrada), el análisis matemático para poder resolver esta ecuación diferencial es
$$A\frac{{y\left( {t\Delta } \right) - y\left( {\left( {t - 1} \right)\Delta } \right)}}{\Delta } + By\left( t \right) = C\frac{{x\left( {t\Delta } \right) - x\left( {\left( {t - 1} \right)\Delta } \right)}}{\Delta } + Dx\left( t \right)$$
$$y[n] = y\left( {t\Delta } \right);x[n] = x\left( {t\Delta } \right)$$
$$A\frac{{y\left[ n \right] - y\left[ {n - 1} \right]}}{\Delta } + By\left[ n \right] = C\frac{{x\left[ n \right] - x\left[ {n - 1} \right]}}{\Delta } + Dx\left[ n \right]$$
$$A\left( {y\left[ n \right] - y\left[ {n - 1} \right]} \right) + B\Delta y\left[ n \right] = C\left( {x\left[ n \right] - x\left[ {n - 1} \right]} \right) + D\Delta x\left[ n \right]$$
$$y[n]\left( {A + B\Delta } \right) - Ay[n - 1] = y[n]\left( {C + D\Delta } \right) - Cy[n - 1]$$
$$y[n] = \frac{{y[n]\left( {C + D\Delta } \right) - Cy[n - 1] + Ay[n - 1]}}{{\left( {A + B\Delta } \right)}}$$
\noindent
Con este análisis y el programa creado, su puede solucionar cual ecuación diferencial, pero el valor de XV es de un vector de $n$ datos.
